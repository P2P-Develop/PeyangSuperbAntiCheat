## 報告方法

脆弱性を見つけたら、\[security\]ラベルを付けて[こ↑こ↓](https://github.com/peyang-Celeron/PeyangSuperbAntiCheat/issues)まで報告してきてね♡
心優しいコラボレーター達や開発者が
> これやったらおもっくそ権限無視できるじゃないですかやだー  
  
と言ったらセキュリティ草案立ててhotfixなりなんなりで急遽修正します。
